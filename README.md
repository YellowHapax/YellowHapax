## Memory as Baseline Deviation (MBD) Framework

I am the author of the Memory as Baseline Deviation framework — a unified relational
account of memory, identity, and cognition grounded in predictive processing.

The framework proposes that memory is not the retrieval of stored traces but the
continuous computation of deviation from an integrated experiential baseline. From
this foundation it derives accounts of childhood amnesia, false memory, memory drift,
and the structure of social and psychological influence.
